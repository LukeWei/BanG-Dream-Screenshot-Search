# Third-Party Notices

## Screenshot catalog

Screenshot metadata and image URLs are derived from:

- Website: <https://ave-mujica-images.pages.dev/>
- Source repository: <https://github.com/serser322/ave-mujica-images>
- Repository license: MIT, covering that project's source code

The bundled catalog contains screenshot names, episode metadata, and remote URLs fixed to a specific upstream commit. It does not contain copies of the upstream WebP or JPG screenshot assets.

At runtime, WebP previews are requested directly from the upstream repository. Only the JPG selected by the user is downloaded to a temporary clipboard file. This extension does not proxy those requests, provide an offline image cache, or include analytics.
